  This demo is based on the GD32190R-EVAL-V1.3 board, it shows how to use DAC channel 
to generate the different voltage on DAC Channel output.The DAC channel output 
pin is configured PA4. DAC Channel is configured to have an output voltage of VREF/2,
measured to be 2.5V by a multimeter.
