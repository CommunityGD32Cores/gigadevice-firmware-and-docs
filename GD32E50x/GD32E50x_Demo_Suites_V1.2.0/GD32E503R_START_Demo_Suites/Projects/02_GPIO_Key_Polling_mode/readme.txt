  This demo is based on the GD32E503R-START-V1.0 board, it provides a description 
of how to use User Key to control the LED2. The demo uses polling mode. Press the 
key, LED2 will be on. Press the Key again, LED2 will be off.

  On the GD32E503R-START-V1.0 board, the User Key connected to PA0 and the LED2 connected 
to PA8.
