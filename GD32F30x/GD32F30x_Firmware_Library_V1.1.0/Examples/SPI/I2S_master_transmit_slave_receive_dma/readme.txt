  This example is based on the GD32F30x-EVAL board, it shows I2S1 master send and I2S2 
slave receive communication use DMA mode. After the communicate is complete, if receive 
data equal to send data, led1 turn on, if not led1 turn off.

  Connect I2S1 WS PIN(PB12) to I2S2 WS PIN(PA15).
  Connect I2S1 CK PIN(PB13) to I2S2 CK PIN(PB3).
  Connect I2S1 SD PIN(PB15) to I2S2 SD PIN(PB5).
