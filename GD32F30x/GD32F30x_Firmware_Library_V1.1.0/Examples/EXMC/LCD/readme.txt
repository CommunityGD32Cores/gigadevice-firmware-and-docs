  This demo is based on the GD32F30x-EVAL board, it shows how to use EXMC 
peripheral to drive LCD. This board uses EXMC_NE3 to support LCD. You can see
GigaDevice logo and website on the LCD screen.
  A23,D0-D15,NWE,NOE and NE3 must be fitted.
