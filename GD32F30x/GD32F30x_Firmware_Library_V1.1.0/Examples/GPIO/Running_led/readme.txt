  This example is based on the GD32F30x-EVAL board, it provides a description
of Running_led. After system start-up, firstly, LED1 on, then, LED2 on, four
LEDs can light periodically.

  On the GD32F30x-EVAL board,LED1 connected to PC6, LED2 connected to PC7, LED3
connected to PC8, LED4 connected to PC9.
