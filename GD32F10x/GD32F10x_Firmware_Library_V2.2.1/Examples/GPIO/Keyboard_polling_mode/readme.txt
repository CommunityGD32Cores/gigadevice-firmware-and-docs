  This example is based on the GD32107C board, it provides a description 
of how to use Tamper key to control the LED2. The example uses polling mode.

  Press the Tamper key, LED2 will be on. Press the Tamper Key again, LED2 will be off.

  On the GD32F107C-EVAL-V1.3 board, the Tamper key is connected to PC13 and the LED2 
is connected to PC0.
