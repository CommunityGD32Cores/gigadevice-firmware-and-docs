  This example is based on the GD32F107C-EVAL board, it provides a description
of Running_led. After system start-up, firstly, LED1 on, then, LED2 on, four
LEDs can light periodically.

  On the GD32F107C-EVAL board,LED1 connected to PC0, LED2 connected to PC2, LED3
connected to PE0, LED4 connected to PE1.
