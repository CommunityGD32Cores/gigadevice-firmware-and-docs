  This example is based on the GD32107C-EVAL board, it provides a description 
of how to use Tamper key to control the LED1. The example uses polling mode.

  Press the Tamper key, LED1 will be on. Press the Tamper Key again, LED1 will be off.

  On the GD32F107C-EVAL board, the Tamper key is connected to PC13 and the LED1 
is connected to PC0.
