  This demo shows how to generate the interrupt request, and toggle the LED. 
After the system start-up, LED1 is on. When Tamper Key is pressed, LED2 is 
toggled, it is switched on; when Tamper Key is pressed once again, LED2 is
toggled, it is switched off.
