  This example is based on the GD32207i-EVAL board, it provides a description 
of how to use Tamper key to control the LED1. The example uses polling mode.

  Press the key, LED1 will be on. Press the Key again, LED1 will be off.

  On the GD32F207i-EVAL board,the Tamper key connected to PC13 and the LED1 
connected to PF6.
