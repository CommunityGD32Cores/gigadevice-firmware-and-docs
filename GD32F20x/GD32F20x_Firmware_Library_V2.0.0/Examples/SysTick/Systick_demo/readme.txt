  This example is based on the GD32207i-EVAL board, it provides a description
of Systick_demo. After system start-up,four LEDs are toggled.
  On the GD32F207i-EVAL board, LED1 connected to PF6, LED2 connected to PF7, LED3
connected to PF8, LED4 connected to PF9.
