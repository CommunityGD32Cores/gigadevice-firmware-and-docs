  This demo is based on the GD32207i-EVAL board, it shows how to use EXMC 
peripheral to control SDRAM memory. 
  This demo shows the write and read data operation process of SDRAM memory
by EXMC module. If the access correctly, LED1 will be turned on. Otherwise,
turn on LED3.
  
note:
  The timing parameters are calculated according to the 120MHz system clock.
