  This demo is based on the GD32207i-EVAL board, it shows how to enable and use 
the color keying functionality.
  After LCD initialization, LCD layer 0 and color keying configuration, an image 
is displayed on LCD.
  Color keying is enabled/disabled on the image when the TAMPER button is pressed.

  Jump JP31 to LCD
