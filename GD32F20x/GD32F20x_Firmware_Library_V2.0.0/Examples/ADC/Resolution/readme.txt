  This demo is based on the GD32207i-EVAL board, it shows how to use the function
of programmble resolusion. In this demo, 6B resolusion is configured. PC3(CHANNEL13)
is chosen as analog input pin. The ADC conversion begins by software, the converted 
data is printed by USART.

  The analog input pin should configured to analog mode.

  Jump the JP5 to USART.
